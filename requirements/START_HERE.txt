⚡ CRITICAL IMPLEMENTATION DETAILS
════════════════════════════════════════════════════════════════════════════

Three gotchas that will trip you up:

1️⃣  INSTANCEDMESH HANDLING (Files: selection.md, sectionplane.md)
    IFC.js uses InstancedMesh. When calculating world space:
    ✓ Must combine mesh.matrixWorld with instance-specific matrix
    ✓ Get instance matrix: mesh.getMatrixAt(instanceId, matrix)
    ✓ Combine: worldMatrix = mesh.matrixWorld × instanceMatrix

2️⃣  CLIPPING PLANE DIRECTION (File: sectionplane.md)
    Three.js clips on POSITIVE side of plane normal:
    ✓ To clip "behind" a face, you must NEGATE the normal
    ✓ clipNormal = normal.clone().negate()

3️⃣  MATERIAL CLONING (Files: selection.md, visibility.md)
    Always clone before modifying to avoid shared material effects:
    ✓ Clone first: mesh.material = mesh.material.clone()
    ✓ Then modify: mesh.material.opacity = 0.5
